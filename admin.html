Cara Menjalankan Backend

1. Install dependencies:

cd sipas/backend && npm install

2. Jalankan server:

node server.js

3. Server berjalan di port 3001 secara default.

4. Set environment variable untuk production:

PORT=3001 ADMIN_TOKEN=password_kuat node server.js
